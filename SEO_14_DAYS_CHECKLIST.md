# SEO чеклист на 14 дней (rybasvprud.ru)

Старт: 21.04.2026

## День 0 (сегодня)

1. Яндекс.Вебмастер: добавить sitemap `https://rybasvprud.ru/sitemap.xml`.
2. Google Search Console: добавить sitemap `https://rybasvprud.ru/sitemap.xml`.
3. Проверить главное зеркало: `https://rybasvprud.ru` (без `www`).
4. Через проверку URL отправить на переобход:
   - `https://rybasvprud.ru/`
   - `https://rybasvprud.ru/ryba/`
   - `https://rybasvprud.ru/blog/`
   - 3 карточки рыбы
   - 3 статьи блога

## Дни 1-3

1. Проверять разделы с индексированием/страницами в обоих кабинетах.
2. Если есть ошибки `Blocked by robots.txt`, `noindex`, `404`, `5xx` — фиксировать и сразу править.
3. Проверять, что новые URL из sitemap начинают появляться в статусах «Обнаружено/Просканировано».

## Дни 4-7

1. Каждый день смотреть:
   - ошибки сканирования;
   - исключенные страницы;
   - канонические URL (нет ли «выбрана другая каноническая» для важных страниц).
2. Повторно отправить на переобход главную, `/blog/`, `/ryba/`, если статус не меняется.
3. Проверить мобильную пригодность в Google Search Console (раздел Core Web Vitals/Удобство для мобильных).

## Дни 8-14

1. Смотреть первую динамику в «Эффективность/Поисковые запросы»:
   - показы;
   - клики;
   - CTR;
   - средняя позиция.
2. Проверить, какие страницы уже дают показы (обычно главная, категории, первые статьи).
3. Для страниц без показов:
   - усилить внутренние ссылки с главной/категорий/статей;
   - проверить title/description/H1 на уникальность и релевантность.

## Что считается нормой в первые 2 недели

1. Индексация идет постепенно, не мгновенно.
2. Часть URL может быть в «Обнаружено, не индексировано» — это нормально на старте.
3. Важнее отсутствие критичных ошибок (`5xx`, массовые `404`, блокировка robots).

## Красные флаги (исправлять сразу)

1. Sitemap не читается или отклоняется.
2. Главная страница не индексируется.
3. Важные URL попадают в `noindex`/`blocked by robots`.
4. Много дубликатов с неправильной канонизацией.

## Мини-отчет (ежедневно 1 раз)

1. Сколько URL отправлено в sitemap.
2. Сколько URL в индексе (Яндекс/Google).
3. Какие новые ошибки появились.
4. Какие страницы отправлены на переобход.
